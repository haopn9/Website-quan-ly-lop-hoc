# Website-Qu-n-l-l-p-h-c-
Đồ án thực tập tốt nghiệp : Xây dựng Website quản lý lớp học

## GIỚI THIỆU
Đề tài hướng tới xây dựng một website quản lý lớp học , tập trung vào module làm việc nhóm , cho phép giảng viên và sinh viên quản lý nhóm học tập một cách tập trung , minh bạch và có thể theo dõi được toàn bộ quá trình làm việc
